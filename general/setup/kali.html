Purpose of the tutorial

This quick guide is for setting up Kali Linux using VirtualBox and would workout to be independent of the Host Operating System ie. Windows, Linux, MacOS

Jargon

Host: The physical hardware running the Virtual Machine is generally referred to as the host
Guest: Virtual machine emulated on the host is generally referred to as the guest

A host can emulate several guests. Each of these guests can emulate different operating systems and hardware platforms.
